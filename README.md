# vue-covid-tracker

This API will get the info of the tracker of https://covid19api.com

## Project setup

```
npm install -g @vue/cli
sudo npm install -g npm@8.6.0
vue add tailwind
npm i moment
```

### Compiles and hot-reloads for development

```
npm run serve
```
